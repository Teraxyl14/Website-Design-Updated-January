## üöÄ Getting Started

Clone the repository and install dependencies to start the "Liquid Engine" locally.

### Prerequisites
*   Node.js (v18+ recommended)
*   npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Teraxyl14/Website-Design-Updated-January.git

# 2. Navigate to project directory
cd Website-Design-Updated-January

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`.
